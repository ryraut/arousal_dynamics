#### Code for time delay embedding and nonlinear (neural network) predictions as performed in [Raut et al. 2023](https://doi.org/10.1101/2023.11.06.565918)
Code presently includes an example jupyter notebook (ipynb) that delay embeds an input scalar time series (pupil) and learns a neural network mapping to a high-dimensional time series (widefield optical images).
Code will be updated to reproduce all major analyses from the paper.
